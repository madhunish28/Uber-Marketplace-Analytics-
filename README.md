# Executive Marketplace Analytics Dashboard (Uber Case Study)

An end-to-end Business Intelligence project that analyzes **103,024+ ride-booking records** to uncover operational bottlenecks, revenue leakage, customer behavior, and marketplace performance using **SQL, Excel, and Tableau**.

This project simulates an executive analytics solution for a ride-sharing platform by transforming raw transactional data into interactive dashboards and actionable business insights.

---

## 📌 Project Objectives

* Analyze ride-booking performance across the marketplace.
* Identify revenue leakage caused by unsuccessful bookings.
* Evaluate vehicle portfolio performance.
* Monitor customer and driver satisfaction.
* Understand operational friction and cancellation patterns.
* Build executive-level dashboards for business decision-making.

---

## 🛠 Tech Stack

* **SQL (MySQL)** – Data modeling, analytical queries, and business views
* **Microsoft Excel** – Data preparation and preprocessing
* **Tableau** – Interactive dashboards and business intelligence
* **GitHub** – Project documentation and version control
